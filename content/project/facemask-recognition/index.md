## **Overview**
This project implements a deep learning-based computer vision system to detect face masks in real time, addressing public safety concerns during the COVID-19 pandemic.

> Enables automated public safety monitoring by detecting mask compliance in real time.
---

## **Problem**
- Low compliance with mask-wearing in public spaces
- Need for automated monitoring without human supervision

---

## **Solution**
I developed a **Convolutional Neural Network (CNN)**model to detect faces and classify mask usage.

---

## **System Design**
### Model
- CNN-based architecture for image classification
- Face detection followed by mask classification

### Processing
Image input → face detection → bounding box → classification

### Tools & Technologies
- Python
- Deep Learning (CNN)
- Computer Vision libraries

---

## **Key Features**
- Real-time face detection and mask classification
- Visual feedback through bounding boxes
- Scalable for deployment in public monitoring systems

---

## **Outcome**
- Successfully trained and tested mask detection model
- Demonstrated feasibility for public safety applications
