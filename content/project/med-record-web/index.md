## **Overview**
This project presents a web-based system designed to centralize medical records across multiple healthcare providers to improve accessibility and efficiency.

> Streamlines healthcare data access and reduces administrative friction across institutions.
---

## **Problem**
- Fragmented medical records across hospitals and clinics
- Continued reliance on paper-based systems
- Inefficiencies in insurance verification processes

---

## **Solution**
I developed a **centralized medical record management system** accessible by authorized institutions.

---

## **System Design**
### Backend
- Database for storing patient records
- Structured data management system

### Frontend
User interface for:
- Registration
- Record management
- Data search and retrieval

### Tools & Technologies
- PHP
- MySQL
- XAMPP
- JavaScript & CSS

---

## **Key Features**
- Centralized patient data system
- Multi-user access control
- Efficient search and update functionality

## **Outcome**
- Fully functional web application prototype
